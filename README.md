##  How to Use This Repository

### Viewing the Code
1. Navigate to any day's folder
2. Open the `.py` file (for Python) or `.html`/`.css` files (for web)
3. Read the comments and code
4. Run the code yourself to see results

### Running Python Code
```bash
# Navigate to repository
cd python-revision

# Run specific day
python day_1/revision.py

# Run specific playlist exercise
python python_100_days_challenge/day_15/script.py
```

### Running HTML/CSS Projects
```bash
# Simply open in browser
# Right-click on .html file → Open with Browser
# Or use VS Code Live Server extension
```

### Cloning the Repository
```bash
# Clone this repo
git clone https://github.com/YOUR_USERNAME/python-revision.git

# Navigate into it
cd python-revision

# View the contents
ls -la
```
